# Aplikasi Pembukuan PT. Sumber Ganda Mekar

Aplikasi penjualan, pembelian, dan sistem akuntansi harian yang terintegrasi dengan Supabase Cloud.

## Fitur
- Auto-sync ke Supabase Cloud.
- Multi-user authentication.
- Laporan Penjualan & Pembelian.
- Jurnal Umum & Buku Besar.

## Deployment
Aplikasi ini dideploy secara otomatis menggunakan GitHub Actions ke GitHub Pages.
